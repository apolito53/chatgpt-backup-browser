What do these mean?

Robust is the better default. It keeps more metadata available, handles attachments more completely, and is what you want unless a huge archive is making the browser struggle.

Lightweight skips some heavier detail during the first pass so very large exports load more easily, but you give up some richness until you load more details later.

Rule of thumb: start with Robust. Switch to Lightweight only if the browser starts acting dramatic.
